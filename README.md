## 🔧 Installation

1. Clone repositori:
   ```sh
   git clone https://github.com/Ester-coder/chat-app.git
   cd chat-app
   ```

2. Install Dependency
   ```sh
   npm install
   ```

## 💡 Usages
1. Run App
   - run server
   ```sh
   cd server
   node index.js
   ```
   - run app
   ```sh
   npm start
   ```
